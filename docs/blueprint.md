# **App Name**: Manoel da Farmacia Platform

## Core Features:

- Smart Patient CRM: Comprehensive patient profile with biological/chronological age tracking, lifestyle markers, and file uploads.
- Integrative Anamnesis Engine: Dynamic multi-step form using the SOAP method, with sections for gut health, sleep, phytotherapy, and nutraceutical history.
- Interactive Treatment Planner: A tool to create integrative protocols linking supplements, essential oils, and floral therapies. Includes the option to add an explanation to be crafted by generative AI using a reasoning tool that can determine whether or not the requested therapies match up well with other notes and plans.
- Financial/Billing Dashboard: Track consultation revenue, pending payments, and cash flow formatted in BRL.
- Appointment Scheduler: Full calendar integration with status management (Scheduled, Confirmed, Completed, No-show).
- LGPD Compliance: Implement Brazilian GDPR (LGPD) compliance patterns, including data encryption masks and privacy-first UI components.
- Prescription PDF Preview: Generate a preview of a pharmaceutical-grade PDF prescription document.

## Style Guidelines:

- Primary color: Forest Green (#2D5A27) for a natural, calming feel.
- Background color: Soft Gray (#F4F7F5) for a clean, medical aesthetic.
- Accent color: Gold (#D4AF37) for highlights and calls to action, conveying sophistication.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, is well-suited for the content of this application.
- Lucide Icons: Consistent, minimalist icons throughout the platform.
- Dashboard: Quick-action tiles for 'New Patient,' 'Quick Prescription,' and 'Today's Schedule.'
- Subtle transitions and animations for enhanced user experience.
